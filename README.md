## Featured Projects

### RoadPulse

Embedded IoT platform for pothole detection and geolocation using STM32, ESP32, MPU6050, and GNSS. The project focuses on real-time road hazard detection, classification, and location tracking for future road-condition mapping applications.

### Secure Automated Dispensing System

Password-protected liquid dispensing platform featuring cup detection, liquid-level monitoring, wireless status reporting, and automated dispensing control using PIC18 firmware.

### Pure Sine Wave Inverter

SPWM-based inverter design focused on low-distortion AC waveform generation, power electronics, switching control, and DC-to-AC power conversion.

### Greenhouse Controller

RTOS-based environmental monitoring and control system featuring sensor integration, automated actuation, and real-time embedded control.
